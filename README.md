### 🐉 Creature Summoner (2026)

A JavaScript console-based summoning simulator that generates a random mythical creature with attributes such as color, element, rarity, and power level. The experience is enhanced with asynchronous delays to simulate a dramatic summoning sequence.

This project demonstrates core JavaScript fundamentals while introducing asynchronous programming concepts like `async/await` and Promises in a practical, interactive way.

**Key Features:**

* Randomized creature generation (creature, color, element, rarity)
* Dynamic power scaling based on rarity
* Timed summoning sequence using async/await
* Conditional messaging based on rarity outcome

**Stack:** JavaScript · Node.js (Console)
